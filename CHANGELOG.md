# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Initial release of RoboCompute SDK
- Client API for renting computational power
- Provider API for sharing resources and earning
- Solana stablecoins (USDC/USDT) integration
- WebSocket support for real-time task monitoring
- Support for Python, Node.js, Go, Rust, C++
- Framework integrations (FastAPI, Flask, Django, Express, NestJS)
- ROS2 integration for robotics
- Comprehensive error handling
- Rate limiting support
- Staking management for providers
- Earnings and payout management
- Resource management (CRUD operations)
- Task management (CRUD operations)
- Wallet balance checking
- Billing history and invoices
- Provider search and discovery
- Node monitoring and metrics
- Reputation system support
- API base domain: robocompute.xyz/api/

### Security
- Wallet signature authentication
- Secure API key management
- Solana blockchain integration
